# Routine Tracking Mobile App

<div class="project-hero">
  <img src="/assets/images/routinier.jpg" alt="Routine tracking app routinier">
</div>

Routinier is a routine tracking application. This is a personal project to build my own iOS app to track routines and good habits. 
Routines can be shared across users and get synchronized. 

## Technical Stack

Built for iOS with SwiftUI and CoreData, featuring CloudKit integration for data synchronization and sharing capabilities.
